# Sleep-Timer
I've always needed an application that would shut down my computer on a timer (for various needs). Of course, you can go to the terminal every time and write “shutdown /s /t x”, but it's not convenient. And so, it just so happened that I've been studying python for a month now, and I had the thought: “Why not make such an application?”. And here I did it.

So far it has no interface and works only in the terminal, but I am already happy with my Sleep Timer v1.0. In the future I plan to add an interface and many different features that will make everyday life easier.

Yes, I realize that there is nothing difficult in writing such a timer, but I hope that still, someone will need it.

Use it. And you can leave me feedback on the code (for example, what can be improved and what can be removed). I'm still starting and any feedback would be helpful.
